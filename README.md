# Site de coach sportif

Un template en HTML et CSS pour mettre en avant le travail d'un coach sportif. Le template, responsive sur mobile, tablette et desktop intègre le framework bootstrap 4 et quelques animations en JavaScript.

Il s'agit d'un exercice réalisé pendant ma formation de développeur web pour s'entraîner à l'intégration, au responsive design et à la programmation front-end.

Le cahier des charges imposait les spécifications suivantes :
- Une page d'accueil avec image fullscreen
- Un formulaire de contact
- Un formulaire de réservation avec présence d'un calendrier
- Un page avec les forfaits proposés
- Une page de présentation du coach
- Au moins une animation JavaScript
- Au moins une vidéo
- Un design 100% responsive
- Une charte graphique cohérente et moderne
